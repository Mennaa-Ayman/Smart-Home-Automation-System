# Smart Home Automation System (using Design Patterns in C++)

##  Project Overview:
This **Smart Home Automation System** involves designing and implementing a modular Smart Home Automation System in C++. The system will simulate managing various smart devices found in a modern home environment, such 
as lights, thermostats, and security cameras. It demonstrates architecture-focused software design using multiple **Creational, Structural, and Behavioral Design Patterns.**
